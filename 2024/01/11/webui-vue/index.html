前端框架汇总-Vue系列【持续更新ing....】

发表于 2024-01-11 分类于 前端框架 本文字数： 1.2k 阅读时长 ≈ 4 分钟

View Design

原名iView，后改名为ViewUI，现在叫ViewDesgn。

基于Vue.js3的企业级UI组件库和中后台系统解决方案。

相关产品信息：
View UI Plus 【开源】基于Vue.js3的开源UI组件库；✅
Admin Cloud  【需要购买授权】企业级多租户权限管理开发平台；
Admin Plus   【需要购买授权】开箱即用的中后台前端解决方案；
iCRUD        【需要购买授权】面向配置的增删改查表格组件；

官方地址：iView / View Design 一套企业级 UI 组件库和前端解决方案 (iviewui.com)

D2Admin

D2Admin是一个完全 开源免费 ✅的企业中后台产品前端集成方案，使用最新的前端技术栈，小于 60kb 的本地首屏 js 加载，已经做好大部分项目前期准备工作，并且带有大量示例代码，助力管理系统快速开发。

官方地址:D2开源组

Vue Admin Work

开箱即用的中后台前端解决方案。

相关产品/版本信息：
Admin Work A    【开源】Vue3/vite/Typescript/Antd；✅
Admin Work Pro  【需要购买授权】Vue3/vite/Typescript/Naive UI；
Arco Work       【需要购买授权】Vue3/vite/Typescript/Arco Design；
Vue Admin Work  【开源】Vue2/Webpack/Javascript/Element UI；✅
Vue Admin Work X【开源】Vue3/Webpack/Typescript/Element Plus；✅
Admin Work      【开源】Vue3/vite/Typescript/Naive UI；✅

官方地址：Admin Work (vueadminwork.com)

Naive UI

一个 Vue 3 组件库，比较完整，主题可调，使用 TypeScript，快…

【开源】✅ 在官网未发现需要购买的内容…..

特点：

比较完整

有超过 80 个组件，希望能帮你少写点代码。

顺便一提，它们全都可以 treeshaking。

主题可调

我们提供了一个使用 TypeScript 构建的先进的类型安全主题系统。你只需要提供一个样式覆盖的对象，剩下的都交给我们。

顺便一提，不用 less、sass、css 变量，也不用 webpack 的 loaders。以及你可以试试主题编辑器。

使用 TypeScript

Naive UI 全量使用 TypeScript 编写，和你的 TypeScript 项目无缝衔接。

顺便一提，你不需要导入任何 CSS 就能让组件正常工作。

快

我尽力让它不要太慢。至少 select、tree、transfer、table、cascader 都可以用虚拟列表。

官方地址：Naive UI: 一个 Vue 3 组件库

SCUI Admin

基于 Vue3、elementPlus 持续性的提供独家组件和丰富的业务模板帮助你快速搭建企业级中后台前端任务。

【开源】✅ 在官网未发现需要购买的内容…..

官方地址：SCUI (gitee.io)

Naive Ui Admin

基于 Vue3、Vite、TypeScript 等最新技术栈开发。

【开源】✅ 在官网未发现需要购买的内容…..

特点：

最新技术栈：基于 Vue3、Vite、TypeScript 等最新技术栈开发。

轻量快速的热重载：无论应用程序大小如何，都始终极快的模块热重载；

最佳实践：常用页面布局，组件使用示例，插件示例实现；

组件封装：对日常使用频率较高的组件二次封装，满足基础工作需求；

主题配置：丰富的主题配置及黑暗主题适配；

权限管理：完善的前后端权限管理方案

官方地址：Naive Ui Admin 开箱即用中台前端框架 | Naive Ui Admin 开箱即用中台前端框架 (naiveadmin.com)

Vben Admin

​ Vue-Vben-Admin 是一个基于 Vue3.0、Vite、 Ant-Design-Vue、TypeScript 的后台解决方案，目标是为开发中大型项目提供开箱即用的解决方案。包括二次封装组件、utils、hooks、动态菜单、权限校验、按钮级别权限控制等功能。项目会使用前端较新的技术栈，可以作为项目的启动模版，以帮助你快速搭建企业级中后台产品原型。也可以作为一个示例，用于学习 vue3、vite、ts 等主流技术。该项目会持续跟进最新技术，并将其应用在项目中。

【开源】✅ 在官网未发现需要购买的内容…..

官方地址：Home | Vben Admin (vvbin.cn)

EleAdmin

通用型后台管理模板，界面美观、开箱即用，拥有丰富的扩展组件和模板页面，适合各类中后台应用。

相关产品/版本信息：
EleAdminPlus 【需要购买授权】使用 Vue3、ElementPlus最新版搭建， 使用当前主流的前端技术栈：Vite、Vue、CompositionApi、Typescript、VueRouter、Pinia、Axios、Scss 等；

EleAdminPro 【需要购买授权】使用 Vue3、AntDesignVue最新版搭建， 使用当前主流的前端技术栈：Vite、Vue、CompositionApi、Typescript、VueRouter、Pinia、Axios、Less 等；

EleAdmin 【需要购买授权】使用 Vue2、ElementUI最新版搭建， 使用当前主流的前端技术栈：Webpack、Vue、VueX、VueRouter、VueCLI、Axios、Scss 等， 并优化了大量的 ElementUI 样式；

官方地址：EleAdmin后台管理模板
