MacOS Sonoma安装Huawei DevEco Studio 3.1.1

发表于 2023-11-17 更新于 2025-06-19 分类于 软件安装 ， 鸿蒙系统 本文字数： 1.4k 阅读时长 ≈ 5 分钟

一、系统环境

操作系统：macOS Sonoma 14.1.1 (23B81)

IDE版本：DevEco Studio 3.1.1

NodeJs版本：v14.21.3

*建议系统硬件配置：内存8G+,建议16G起步.

二、IDE简介

HUAWEI DevEco Studio（获取工具请单击链接下载，以下简称DevEco Studio）是基于IntelliJ IDEA Community开源版本打造，为运行在HarmonyOS和OpenHarmony系统上的应用和服务（以下简称应用/服务）提供一站式的开发平台。

作为一款开发工具，除了具有基本的代码开发、编译构建及调测等功能外，DevEco Studio还具有如下特点：

  • 高效智能代码编辑：支持ArkTS、JS、C/C++等语言的代码高亮、代码智能补齐、代码错误检查、代码自动跳转、代码格式化、代码查找等功能，提升代码编写效率。更多详细信息，请参考编辑器使用技巧。
  • 低代码可视化开发：丰富的UI界面编辑能力，支持自由拖拽组件和可视化数据绑定，可快速预览效果，所见即所得；同时支持卡片的零代码开发，降低开发门槛和提升界面开发效率。更多详细信息，请参考使用低代码开发应用/服务。
  • 多端双向实时预览：支持UI界面代码的双向预览、实时预览、动态预览、组件预览以及多端设备预览，便于快速查看代码运行效果。更多详细信息，请参考使用预览器预览应用/服务界面效果。
  • 多端设备模拟仿真：提供HarmonyOS本地模拟器，支持手机等设备的模拟仿真，便捷获取调试环境。更多详细信息，请参考使用模拟器运行应用/服务。

三、前置说明

​ 对于这个IDE，笔者安装过几次，过程一言难尽，有成功有失败，失败的原因也是各种情况。这次安装是在重装完系统后重新安装该IDE，笔记用于记录安装过程。

​ HUAWEI DevEco Studio截止到当前日期，最新release版本是DevEco Studio 3.1.1版本，4.0是beta版，因为可能存在未知bug，保险期间，建议安装3.1.1版本，本文介绍的也是安装3.1.1版本。

​ IDE的下载到官网进行下载即可：HUAWEI DevEco Studio和SDK下载和升级 | HarmonyOS开发者

​ 需要注意的是，DevEco Studio每个版本对NodeJS的版本是有要求的，这个之前我也没有注意到，在执行过程中总是会碰到nodejs问题。一次偶然的机会在华为开发者论坛找到DevEco Studio SDK与NodeJS版本配套关系：SDK配套关系&适配历史工程-华为开发者论坛 | 华为开发者联盟 (huawei.com)，由于DevEco Studio版本在持续更新，SDK版本也在更新，构建插件版本也在更新，他们必须匹配，才能正常使用。

​ 说明中，最新的IDE版本只列至3.1.0.501，对于3.1.1版本并没有说明，我们也参照这个版本安装nodejs，本文采用的是NodeJs v14.21.3版本。

​ 为保证安装顺利，强力建议安装与之匹配的NodeJs版本，不然出的问题足以让你崩溃。

​ 推荐一个NodeJs多版本可视化管理工具，跨平台使用，可以灵活切换NodeJs版本，点击 跳转下载 可下载最新版本：

四、安装

4.1下载

​ 根据系统环境下载对应版本，本系统是Intel的Mac系统，需要下载Mac(X86)版本，如果是M1/2/3的CPU，需要下载ARM版本。

4.2安装

​ 下载完成后解压，双击打开.dmg文件文件，按要求拖动APP至Applications文件夹。

4.3配置

​ 在启动台点击DevEco-Studio图标打开IDE，启动后，弹出配置对话框。图中①所示是指安装Ohpm的位置，一般情况都是未先行安装的，勾选Install选项，安装位置无需特意改变（如果本地已经安装配置的话，系统会自动检测本地路径，则无需选择Install），确认无误，点击“Next”进入下一步。
